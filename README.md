# cog-svcs_language

POC code for running Azure Cognitive Services language analysis

Specifically code is set up to look at a law enforcement scenario where we are interested in POL data 
(People, Organisations, Locations) which are often in large blocks of unstrcutred data submitted 
by memebers of the public

Pre-reqs:

azure-ai-textanalytics

Tested on Python 3.9
